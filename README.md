# mi_bear.github.io
El presente mapa interactivo de la biota edáfica de Argentina, presenta toda la información de Clitellata (lombrices), Acari (ácaros) y Colembola (Colémbolos), disponible en publicaciones y comunicaciones científicas. Los puntos señalan la ubicación georreferenciada de cada sitio de muestreo de fauna edáfica.
